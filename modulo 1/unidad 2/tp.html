Tipo Ejemplo de uso Descripcion
button Define un boton clickeable(por lo general se usa para activar scripts de java script)
checkbox Define una checkbox
color Define un selector de colores
date Define un selector de fecha(anio, mes, dia(no se puede hora))
datetime-local Define un selector de fecha y hora(anio, mes, dia, hora(sin zona horaria))
email Define un capo para una direccion de mail
file Define un selector de archivos y un boton para buscar archivos para subirlos
hidden Define una espacio de input escondido
image Define una imagen como el botton de enviar formulario
month Define el dia y el anio(sin zona horaria)
number Define un campo para entrar numeros
password Define un campo para cargar contrasenias
radio Define un boton de radio
range Define un contorlador de rango(como una barra deslizadora)
reset Define un boton de reset
search Define un campo de texto para entrar un string de texto
submit Boton que envia formulario
tel Define un campo para entrar un numero de telefono
text Opcion predefinida del type, Define una campo para una unica linea de texto
time Define un control para entrar un tiempo(sin zona horaria)
url Define un campo para ingresa una url
week Define un control para seleccionar semana y anio
